# Click Me ⚡

A React + Vite starter project with a click counter, theme toggle, and more.

## Features

- ⚛️ Click counter with reset
- 🌙 Dark / Light theme toggle
- ⚡ Powered by Vite HMR

## Getting Started

```bash
npm install
npm run dev
```

## Tech Stack

- [React](https://react.dev/)
- [Vite](https://vite.dev/)
